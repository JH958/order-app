# 메뉴 이미지 저장 폴더

이 폴더에 각 메뉴별 이미지 파일을 저장하세요.

## 이미지 파일 명명 규칙
- 파일명: 메뉴명과 동일하게 저장 (예: `아메리카노.jpg`, `카페라떼.png`)
- 권장 형식: JPG, PNG, WebP
- 권장 크기: 800x600px 또는 16:9 비율

## 예시 파일명
- `아메리카노.jpg`
- `카페라떼.jpg`
- `카푸치노.jpg`
- `카페모카.jpg`
- `카라멜 마키아토.jpg`
- `바닐라 라떼.jpg`
- `자몽에이드.jpg`
- `레몬에이드.jpg`
- `밀크티.jpg`
- `청귤에이드.jpg`
- `유자차.jpg`
- `생강차.jpg`
- `아이스크림.jpg`
- `크로와상.jpg`
- `마카롱.jpg`
- `베이글.jpg`
- `머핀.jpg`
- `쿠키.jpg`

## 사용 방법
1. 이미지 파일을 이 폴더(`ui/public/images/`)에 저장
2. 데이터베이스의 menus 테이블에서 각 메뉴의 image 필드를 업데이트
   - 경로 형식: `/images/아메리카노.jpg`
   - 또는 전체 URL: `http://localhost:5173/images/아메리카노.jpg`

## 참고
- Vite에서는 `public` 폴더의 파일이 루트 경로(`/`)로 서빙됩니다
- 따라서 `public/images/아메리카노.jpg`는 `/images/아메리카노.jpg`로 접근 가능합니다
